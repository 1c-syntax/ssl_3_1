Предназначена для настройки REST-сервиса для обмена со сторонними программами.
Для работы с REST-сервисом не рекомендуется использовать какие-либо уже существующие учетные записи с целью защиты данных программы от взлома, т.к. REST-сервис предназначен для работы со сторонними программами.

Состоит из двух вкладок:
Авторизация - можно ввести имя пользователя и пароль для работы с REST-сервисом;
Состав - можно определить, какие данные будут участвовать в работе автоматического REST-сервиса.

Авторизация

Включите флажок Создать для использования автоматического REST-сервиса отдельные имя пользователя и пароль (рекомендуется).
Заполните поля:
  • Имя пользователя;
  • Пароль;
  • Подтверждение пароля.

Состав

Определите, какие данные будут участвовать в работе автоматического REST-сервиса. 

  • По умолчанию список пуст. Нажмите кнопку Загрузить метаданные, для того чтобы отобразить данные программы.
  • С помощью флажков предоставьте доступ к основным данным, которые будут участвовать в работе REST-сервиса. 
  • После предоставления доступа к основным данным программа выводит список подчиненных (зависимых) данных, без которых работа с основными данными невозможна. Подтвердите возможность их использования.
  • После предоставления доступа нажмите кнопку Сохранить и закрыть.
