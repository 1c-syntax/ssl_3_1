Предназначена для очистки данных вручную, когда при реструктуризации произошла ошибка "Записи регистра стали неуникальными", например:

Открывается с помощью ссылки Очистка устаревших данных в разделе Администрирование - Обслуживание - Обновление программы.

Очистка устаревших данных:

  • запускает длительную операцию поиска устаревших данных; 
  • выводит таблицу с устаревшими данными, которую можно удалить в режиме 1С:Предприятия.

С помощью флажка Вывести количество выводятся в таблицу:

  • количество записей, которое будет удалено в зависимых регистрах сведений;
  • количество регистраторов, записи которых будут удалены в регистрах сведений подчиненных регистратору;
  • количество объектов, которые будут удалены в ссылочных таблицах (справочниках, документах и т.д.).

С помощью флажка Чистить удаляемые выводится список всех устаревших данных (а не только тех, которые могут вызвать ошибку).

Кнопка План очистки открывает форму с описанием, в которой подробно указано в каких таблицах и по каким условиям будет выполняться поиск устаревших данных. 

С помощью кнопки Обновить можно актуализировать данные для очистки. Для того чтобы очистить данные нажмите кнопку Очистить.

После очистки устаревших данных можно продолжить обновление программы в Конфигураторе.
