Форма обработчика

Форма предназначена для редактирования описания обработчика обновления.
Более подробно о правилах формирования описания обработчика и назначение флагов описаны в документации по БСП, раздел Обновление версии ИБ.

Описание

Указываются общие сведения об обработчике обновления.
Флаг Технический проект позволяет добавить обработчик в список обработчиков технического проекта.
Список обработчиков техничекого проекта обладает следующими свойствами:

  • список хранится в неверсиониремом файле каталога .setting проекта git-репозитория
  • обработчики списка выделены условным оформлением в общем списке обработчиков
  • доступен быстрый отбор обработчиков списка

Объекты

Указывается информация о читаемых, изменяемых объектах обработчика.
Дополнительно можно указать какие объекты необходимо блокировать пока обработчик не будет выполнен.
В контекстном меню табличных частей Читаемый объект и Изменяемый объект доступны вспомогательные команды для заполнения этих табличных частей.

Конфликты

После записи обработчика на закладке Конфликты будут выведены все пересечения по читаемым/изменяемых данным с другими обработчиками.
Для каждого возникшего конфликта необходимо указать приоритет (очередность) выполнения обработчика.
