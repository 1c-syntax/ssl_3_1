Описание обработчиков обновления

Обработка предназначена для формирования и редактирования описания обработчиков обновления.
Правила формирования описания обработчиков описаны в документации по БСП, раздел Обновление версии ИБ.

Порядок работы с описанием обработчиков

1. Загрузить описания обработчиков из кода конфигурации.
2. Отредактировать описание существующих обработчиков или добавить описание новых обработчиков обновления.
3. Проконтролировать построение очереди выполнения обработчиков обновления.
4. Сохранить изменения в файлы git-репозитория конфигурации.
