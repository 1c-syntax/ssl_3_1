Предназначена для очистки конфиденциальной информации. Доступна только администратору в меню Все функции.

Предоставляет возможности:

  • Возможность очистки реквизитов;
  • Поддержка управляемых форм;
  • Поддержка бизнес-процессов;
  • Предупреждение при изменении версионируемого объекта.

Очистка конфиденциальной информации производится в двух режимах.

Автоматический режим

  • Применяется при подготовке ИБ для передачи в техническую поддержку.
  • Внимание! Обработка всех данных информационной базы. Рекомендуется производить на копии рабочей базы, т.к. это приведет к безвозвратной потере данных.
  • Нажмите кнопку Выполнить для перехода к обработке данных.
  • По ссылке Полные возможности можно перейти в другой режим.

Полные возможности

  • Для пореквизитной настройки списка обрабатываемых реквизитов необходимо перейти по ссылке Полные возможности.
  • После установки всех параметров обработки нажмите Выполнить.
  • Можно загрузить настройки из файла. Нажмите кнопку , укажите путь к файлу с настройками.
  • Настройки можно сохранить в файл с помощью кнопки . Укажите путь для сохранения файла с настройками.
  • Список обрабатываемых реквизитов состоит из двух вкладок.

Обрабатываемые объекты

Список состоит из двух колонок:

  • В колонке Объект можно с помощью флажков отметить, какие объекты и их реквизиты нужно обработать.
  • В колонке Правило обработки указано По умолчанию (такое значение не выбирается, выставлено программой автоматически, для каждого поля может быть установлено одно из следующих значений).
  • Выберите, как будет обрабатываться конкретный реквизит, с помощью кнопки : Не изменять, Случайное значение, Имя поля и индекс, Имя объекта и индекс. 

Правила обработки

  • Строки: "Не изменять", "Очистить", "Случайное значение", "ИмяПоля+Индекс", "ИмяОбъекта+Индекс".
  • Числа: "Случайное значение", "Не изменять", "Очистить", "Умножить/Разделить/Прибавить/Вычесть".
  • Булево: "Не изменять", "Инвертировать".
  • Даты: "Случайное значение", "Не изменять", "Прибавить/Вычесть(дней)", "Очистить".
  • Хранилище значений: "Не изменять", "Очистить".

С помощью ссылки Обычный вид можно перейти в автоматический режим.

Обезличивание персональных данных

  • После окончания действия соглашения об обработке персональных данных необходимо их очищать.
  • Реализован программный интерфейс для уничтожения персональных данных по переданному отбору;
  • Регламентное задание для автоматического уничтожения данных по истечению срока согласия на предоставление персональных данных.
