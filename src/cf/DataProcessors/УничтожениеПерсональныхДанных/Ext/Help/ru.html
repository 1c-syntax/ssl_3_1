Предназначена для уничтожения персональных данных. 

В соответствии с требованиями, утвержденными Роскомнадзором, оператор персональных данных обязан уничтожать данные в случаях:
  • истечение срока действия согласия субъекта или его отсутствие;
  • незаконного получения персональных данных или отсутствие необходимости в этих данных;
  • неправомерная обработка персональных данных;
  • достижение цели обработки персональных данных;
  • отзыв субъектом персональных данных согласия на обработку.

Для того чтобы была возможность Уничтожать персональные данные, в разделе Администрирование - Настройки пользователей и прав - Защита персональных данных нужно установить флажок Уничтожать персональные данные. 

В верхней части панели указаны субъекты, у которых истек или истекает Срок хранения и требуется провести их дальнейшее уничтожение. 

С помощью флажка Показывать только с истекшим сроком хранения отобразятся те персональные данные,  у которых срок хранения уже истек. 

В поле Комментарий можно увидеть причину, по которой необходимо составить Акт об уничтожении персональных данных субъекта, например Отозвано согласие на обработку персональных данных. С помощью кнопки Создать акт можно составить Акт об уничтожении персональных данных.
