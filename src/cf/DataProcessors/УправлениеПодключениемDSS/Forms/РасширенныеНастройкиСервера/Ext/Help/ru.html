Содержит дополнительные настройки сервера облачной подписи и позволит администратору ограничить
список доступных пользователю способы авторизации в сервисе облачной подписи.

  • В поле представление указывается наименование сервера, оно будет отображено на списках.
  • В поле таймаут указывается время ожидания ответа от сервиса. 
  • В поле шаблон подтверждения указывает имя шаблона, как она было зарегистрирована сервере облачной подписи.
  • В поле версия API необходимо для определения способов подключения к серверу облачной подписи. 
  • Флажок используется позволяет регулировать доступность данного элемента справочника в списках выбора.

Первичная авторизация

  • Пометка включает строку в список доступных способов.
  • В поле Наименование выводится наименование способа. 

Вторичная авторизация

  • Пометка включает строку в список доступных способов поле.
  • В поле Наименование выводится наименование способа.
