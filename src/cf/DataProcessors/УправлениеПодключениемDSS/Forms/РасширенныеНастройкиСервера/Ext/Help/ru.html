Содержит дополнительные настройки сервера DSS и позволит администратору ограничить список доступных пользователю способов авторизации в сервисе DSS.

    • В поле представление указывается наименование сервера, оно будет отображено на списках.
    • В поле таймаут указывается время ожидания ответа от сервиса. 
    • В поле шаблон подтверждения указывает имя шаблона, как оно было зарегистрировано на сервере DSS.
    • В поле версия API необходимо для определения способов подключения к серверу DSS. 
    • Флажок используется позволяет регулировать доступность данного элемента справочника в списках выбора.

    Первичная авторизация

    • Пометка включает строку в список доступных способов.
    • В поле Наименование выводится наименование способа. 

    Вторичная авторизация

    • Пометка включает строку в список доступных способов.
    • В поле Наименование выводится наименование способа.
