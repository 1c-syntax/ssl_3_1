Предназначена для вторичного подтверждения операций в сервис облачной подписи в случае, если отсутствует интернет соединение между сервером облачной подписи и мобильным устройством.

Порядок работы

  • В поле сертификат указывается информация о сертификате, используемый при операции.
  • В поле способ подтверждения можно изменить текущий способ на любой доступный в списке выбора. 
  • В поле документы указывается список документов, с которыми проводится операция. Доступно только для операции создание электронной подписи.
  • В поле графический код отображается QR-code, который необходимо сканировать с помощью камеры мобильного устройства.
  • В поле код необходимо ввести одноразовый пароль, отображенный в мобильном приложении
  • В нижней части выводиться время, оставшееся для окончания процедуры подтверждения операции
  • Также происходит фоновый опрос сервера на наличие подтверждения операции, если появилась связь между мобильным приложением и сервером облачной подписи
