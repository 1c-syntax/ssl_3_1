Предназначена для хранения индивидуальных настроек подключения привязанных к сервису облачной подписи.

Информация учетной записи позволяет хранить данные необходимые для авторизации пользователя и настроек вторичного подтверждения. Техническая информация о подключении к сервису облачной подписи храниться в данных связанного с этой учетной записью элементе сервера облачной подписи.

Настройка авторизации учетной записи

  • В поле сервер указывается элемент справочника серверы облачной подписи, к которой привязывается учетная запись.
  • В поле логин указывается уникальное имя учетной записи, выданного провайдером пользователю при регистрации.
  • В поле первичная авторизация необходимо выбрать способ авторизации пользователя, который допускается при работе с сервисом. Информацию о способах авторизации предоставляет провайдер сервиса.
  • В поле пароль можно опционально указать пароль при первичной авторизации. Введенный пароль сохраняется в настройках текущего пользователя.
  • Поле сертификат необходимо при авторизации по сертификату авторизации, который выпустил провайдер сервиса при регистрации пользователя. Сертификат авторизации это файл в формате pfx содержащий закрытый ключ и сертификат к нему. Поле доступно для заполнения при выборе способа первичной авторизации: сертификат авторизации.
  • Поле пин-код можно заполнить и сохранить в настройках текущего пользователя, если используется способ первичной авторизации: сертификат авторизации. Этот пин-код должен совпадать с паролем, который был задан при создании файла pfx
  • В поле подтверждение операций задается способ, который будет автоматически заполняться при процедуре подтверждения операций подписи, расшифровки и прочих.
  • По ссылке личный кабинет запускается браузер со страницей личного кабинета учетной записи.
  • Комментарий содержит произвольную текстовую информацию об учетной записи облачной подписи.
  • Также можно проверить возможность подключения к серверу с помощью кнопки Проверить подключение
  • С помощью кнопки Состояние учетной записи отображается детальная информация об учетной записи, полученной с сервиса облачной подписи
  • Пункт меню Еще – Проверка работоспособности можно не только проверить возможность подключения к сервису, но дополнительно проверить выполнения базовых операций выполняемых сервисом: подписание, расшифрование, расшифрование и прочие операции
