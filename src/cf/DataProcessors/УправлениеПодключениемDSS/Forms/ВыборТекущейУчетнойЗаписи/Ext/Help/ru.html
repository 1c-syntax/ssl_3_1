Предназначена для выбора текущей учетной записи для работы с электронной подписью сервиса DSS.

В списке выводится:

  • Наименование - полное наименование учетной записи; 
  • Комментарий – дополнительное описание учетной записи; 
  • Авторизован – текущий статус авторизации учетной записи.

 

Если уже происходила авторизация с помощью учетной записи, то такая запись выделяется цветом.

Выбор учетной записи

  • Выделите нужную строку с учетной записью, нажмите Выбрать. Также можно воспользоваться двойным щелчком мыши.

Просмотр данных об учетной записи

  • С помощью команды Изменить контекстного меню по правой кнопке мыши или меню Еще можно открыть данные об учетной записи.
