Отображает описание ошибки возникшей при работе с сервисом облачной подписи.

Информация об ошибке

  • В поле код ошибки указывается числовой код ошибки для ее идентификации.
  • В поле описание указывается описание ошибки. 
  • Ссылка Перейти в журнал регистрации открывает журнал регистрации с отбором по ошибками возникших при работе сервиса облачной подписи.
