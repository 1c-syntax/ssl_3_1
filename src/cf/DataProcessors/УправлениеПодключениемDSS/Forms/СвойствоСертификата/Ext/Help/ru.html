Предназначена для вывода сведений о сертификате, хранящемся в сервисе DSS.

Отображается информация о сертификате, которую предоставил сервер. Основные данные о сертификате:

  • В поле кому выдан указывается информация о владельце сертификата.
  • В поле кем выдан указывается информация об издателе сертификата.
  • В поле действителен указывается дата начала и окончания действия сертификата.
  • В поле отпечаток отображает значение отпечатка в HEX представлении.

Действия с сертификатом

  • Кнопка проверить позволяет проверить валидность сертификат с помощью сервиса проверки.
  • Кнопка выгрузить в файл выгружает содержимое сертификата в файл в формате ASN. 
  • Если при использовании сервиса DSS допускается применение пин-кодов для сертификатов, то по ссылке изменить можно инициировать процесс смены, сброса или установки пин-кода.
