Предназначен для выбора элемента справочника учетной записи DSS.

В списке выводится:

  • Наименование  – наименование учетной записи в БД;
  • Логин – имя учетной записи, под которым будет происходить авторизация; 
  • Сервер – данные о сервере DSS; 
  • Первичная авторизация – выбранный способ авторизации в сервисе;
  • Комментарий  – дополнительное описание учетной записи;

Добавление учетной записи

    • Нажмите Создать.

Просмотр данных об учетной записи

  • С помощью команды Изменить контекстного меню по правой кнопке мыши или меню Еще можно открыть данные об учетной записи.
