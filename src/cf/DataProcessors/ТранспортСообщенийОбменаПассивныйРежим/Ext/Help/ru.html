При таком варианте подключения эта программа отправляет и принимает данные через Интернет только по запросу от программы-корреспондента. 

Данный вид подключения не требует никаких настроек.

С его помощью можно продолжить настройку синхронизации, которая была начата в другой программе. Для этого требуется конфигурационный файл с настройками синхронизации данных через EnterpriseData из программы-корреспондента.
