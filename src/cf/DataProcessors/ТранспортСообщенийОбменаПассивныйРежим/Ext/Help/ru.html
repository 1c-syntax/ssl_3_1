При таком варианте подключения это приложение отправляет и принимает данные через Интернет только по запросу от приложения-корреспондента.

Данный вид подключения не требует никаких настроек.

С его помощью можно продолжить настройку синхронизации, которая была начата в другом приложении. Для этого требуется конфигурационный файл с настройками синхронизации данных через EnterpriseData из приложения-корреспондента.
