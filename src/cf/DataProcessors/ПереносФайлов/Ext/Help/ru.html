Предназначена для переноса файлов в том хранения большего размера или для создания тома хранения архивных файлов. Для редких случаев, когда необходимо отказаться от хранения файлов в томах на устройстве, с помощью этой же команды можно перенести файлы из томов на устройстве в информационную базу. 

С помощью переключателя можно указать, куда перенести файлы (недоступно, если файлы хранятся в информационной базе):

  • Между томами на сервере;
  • Из информационной базы в тома на сервере;
  • Из томов на сервере в информационную базу.

Настройки

  • В указанный том - выберите том из списка, например, Основной том; 
  • Из указанного тома - выберите том из списка, например, Резервный том.
  • Файлы указанного владельца - можно перенести только часть файлов, например, файлы, присоединенные к элементам списка Демо:Организации. Для этого выберите владельца из списка. 
  • Созданные - укажите дату, для того чтобы перенести файлы только за указанный период.
  • С указанным расширением - выберите тип файлов для переноса (изображения или электронные документы).
  • По дополнительным условиям, например, Автор равно Администратор - с помощью этого условия будут перенесены только файлы, автором которых является только администратор. 

При смене способа хранения файлов автоматический перенос файлов из приложения в тома и обратно не происходит. 

Перенос файлов занимает некоторое время, которое зависит от их объема.
