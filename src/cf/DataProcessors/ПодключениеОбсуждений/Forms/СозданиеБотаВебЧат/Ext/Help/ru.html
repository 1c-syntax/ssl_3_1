Подключение бота для веб-сайта

Данный тип подключения предназначен для обеспечения возможности общения посетителей веб-сайта с пользователями учетной системы.

Инструкция по настройке:


1. Настроить (при необходимости) на соответствующей вкладке параметры отображения
2. Нажать Активизировать
3. После успешной активации поле Точка подключения будет заполнено.
4. Обратиться к техническому специалисту, ответственному за сопровождение сайта, сообщить ему Точку подключения. При необходимости предоставить файл с примером веб-страницы. Файл с примером можно сохранить, нажав на гиперссылку Пример веб-страницы.

Параметры отображения.


Видеозвонки - возможность совершать видеозвонки. Допустимые значения:

Доступны - позволяет совершать видеозвонки в чат и из чата
Доступны только в чат - позволяет совершать видеозвонки в чат, но запрещает совершать видеозвонки из чата
Недоступны - запрещает совершать видеозвонки в чат и из чата

Ключ подписи - ключ подписи, который используется для сопоставления пользователя чата и системы взаимодействия. Необязательный параметр.

Местоположение чата - положение свернутого окна с чатом относительно окна страницы. Допустимые значения:

Слева
Снизу
Справа

Выравнивание чата - вертикальное положение свернутого окна с чатом. Допустимые значения:

Сверху
По центру
Снизу

Ориентация кнопки чата в мобильной версии - положение кнопки чата в мобильной версии. Допустимые значения:

Справа снизу
Слева снизу

Цветовая тема - Цветовая схема окна с чатом. По умолчанию используется такая же тема, что и в веб-браузере. Допустимые значения:

Авто
Светлая
Темная

Текст заголовка чата - текст, который будет отображаться в заголовке чата. Необязательный параметр.

Цвет фона заголовка чата - фоновый цвет заголовка чата.

Цвет текста заголовка чата - цвет текста заголовка чата.

Показывать аватары - отображение картинок пользователей в чате. Допустимые значения:

Да
Нет

Состояние камеры пользователя приложения по умолчанию - включает или выключает камеру клиентского устройства при начале видеозвонка

Код языка чата - язык локализации чата (включая служебные сообщения).
