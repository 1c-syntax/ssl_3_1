Настройки предназначены для организации обсуждений с помощью внешних приложений и мессенджеров.

С помощью кнопки Создать выберите и настройте нужный чат. Для каждого типа чата предусмотрены свои параметры подключения.

Чат Max

  • Для получения сообщений из Max необходим бот, с которым будут общаться ваши контакты.
  • Если у вас в Max еще нет бота, создайте его на платформе Max для партнеров для вашей организации. 
  • Перед настройкой рекомендуется ознакомиться с инструкциями:
    • Как создать профиль организации;
    • Как создать бота и получить токен.
  • Заполните поля настройки чата:
    • Наименование — произвольное название бота в Max для отображения в списке;
    • Токен — ключ доступа, выданный на платформе Max для партнеров после создания бота.
  • В поле Кто получает и отправляет сообщения от лица бота Max с помощью кнопки Подобрать выберите пользователей приложения, которые будут работать с сообщениями из Max.
  • После заполнения всех данных нажмите кнопку Активировать для включения интеграции.

Чат Telegram

  • Перейдите в Telegram. Для создания бота напишите пользователю @BotFather и следуйте его инструкциям.
  • Подробные настройки бота описаны в официальной документации: Боты: информация для разработчиков.
  • Выберите боту Имя и произведите необходимые настройки в Telegram.
  • После успешного создания бота укажите Наименование бота и Токен (полученный от @BotFather).
  • С помощью кнопки Подобрать выберите из списка пользователей, которые будут получать и отправлять сообщения в Telegram.

Чат Вконтакте

  • Перейдите в социальную сеть Вконтакте.
  • Следуя указаниям приложения, создайте сообщество (группу).
  • Выберите сообществу Имя и произведите необходимые настройки во Вконтакте (в разделе "Работа с API").
  • После успешного создания чата заполните поля:
    • Наименование — название чата в программе;
    • Ключ — ключ доступа сообщества;
    • Ключ группы — идентификатор группы.
  • С помощью кнопки Подобрать выберите из списка пользователей, которые будут получать и отправлять сообщения в приложении.
  • Нажмите кнопку Активировать.

Чат WhatsApp

  • Предварительно заключите договор с провайдером WhatsApp Business API — компанией "Девино Телеком".
  • Подробная инструкция по подключению доступна на официальном сайте: Инструкция по подключению к WhatsApp Business API через провайдера «Девино Телеком».
  • После заключения договора получите у менеджера "Девино Телеком":
    • Ключ (token);
    • Имя канала.
  • После настройки интеграции необходимо передать навигационную ссылку точки подключения менеджеру "Девино Телеком".
  • Заполните поля:
    • Наименование — название чата в программе;
    • Ключ — полученный токен доступа;
    • Имя канала — название канала WhatsApp.
  • Точка подключения заполняется автоматически.
  • С помощью кнопки Подобрать выберите из списка пользователей, которые будут получать и отправлять сообщения от лица аккаунта WhatsApp.

Чат на сайте

Чат можно интегрировать с корпоративным сайтом или интернет-магазином, что обеспечит непрерывность коммуникации и автоматизирует общение с посетителями даже в нерабочее время благодаря использованию бота. Подробная документация доступна на ИТС: Руководство разработчика. Система взаимодействия.
  • На вкладке Основное заполните поле Наименование.
  • Точка подключения заполняется автоматически после успешной активации.
  • С помощью кнопки Подобрать выберите из списка пользователей, которые будут получать и отправлять сообщения.
  • Настройте (при необходимости) на соответствующей вкладке Параметры отображения, например, внешний вид, расположение окна чата и т.д.
  • Нажмите кнопку Активировать.
  • Обратитесь к техническому специалисту, ответственному за сопровождение сайта, сообщите ему Точку подключения.

Интеграция Webhook

Webhook — это механизм, который позволяет различным сервисам и приложениям обмениваться данными в режиме реального времени. Интеграция с Webhook обеспечивает одностороннюю коммуникацию от внешнего приложения в Систему взаимодействия. С помощью интеграции можно создать новые сценарии взаимодействия, например, информировать пользователей о новых заказах через сайт интернет-магазина. Подробнее см. в документации на ИТС: Руководство разработчика. Интеграция с Webhook.
  • Заполните Наименование бота в 1С.
  • С помощью кнопки Подобрать выберите из списка пользователей, которые будут получать сообщения.
  • Нажмите кнопку Активировать.
  • Точка подключения заполняется автоматически после успешной активации.
  • Обратитесь к техническому специалисту, ответственному за сопровождение сайта, сообщите ему Точку подключения.
Если все было корректно настроено, в программу должны приходить сообщения из подключенных внешних приложений.
