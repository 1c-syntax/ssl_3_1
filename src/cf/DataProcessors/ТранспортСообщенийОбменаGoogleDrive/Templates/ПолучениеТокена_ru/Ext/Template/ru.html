1. В консоли разработчика Google на странице управления ресурсами создайте новый проект по кнопке CREATE PROJECT (создать проект).


2. На странице создания проекта укажите Project name (Имя проекта), или оставьте по умолчанию. Нажмите кнопку CREATE (создать).

3. Перейдите на страницу библиотеки и выберите Google Drive API.


4. При помощи кнопки ENABLE активируйте Google Drive API.


5. Перейдите в раздел AOuth consent screen. Выберите External и нажмите CREATE.


6. На следующей странице заполните поля App name (имя приложения), user support email, email address и нажмите SAVE AND CONTINUE.


7. Следующие шаги можно пропустить, нажимая SAVE AND CONTINUE.

8. Перейдите по ссылке, сделайте данные публичными, кнопка PUBLISH APP.

9. Перейдите в раздел credentials и нажмите кнопку CREATE CREDENTIALS – OAuth client ID.


10. Выберите тип приложения Desktop App и нажмите CREATE.


11. Скопируйте Client ID и Client secret в соответствующие поля внизу формы и нажмите Получить токен.


12. В открывшемся окне пройдите авторизацию в сервисе google и дайте разрешение на использование сервиса.
