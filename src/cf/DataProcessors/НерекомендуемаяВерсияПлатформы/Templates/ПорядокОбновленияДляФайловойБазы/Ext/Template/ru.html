Порядок обновления на новую версию 1С:Предприятия 8.3

1. Скачать дистрибутив 1С:Предприятия 8.3

Скачайте дистрибутив последней версии платформы 1С:Предприятия 8.3 в сервисе 1С:Обновление программ. При необходимости укажите логин и пароль.

Далее следует выбрать актуальную (последнюю официально опубликованную) версию Технологической платформы 8.3.

Для скачивания файла нажмите на гиперссылку, которая соответствует вашей операционной системе:

  • Технологическая платформа 1С:Предприятия для Windows
  • Технологическая платформа 1С:Предприятия для Linux
  • Клиент 1С:Предприятия для macOS

Далее по ссылке Скачать дистрибутив откроется диалог сохранения файла на компьютер. Запомните папку, в которую его сохранили.

2. Установить новую версию 1С:Предприятия 8.3

Если у вас операционная система Microsoft Windows, то обратите внимание: для установки программы пользователь Windows должен обладать правами администратора. Распакуйте загруженный файл с архивом. В папке, в которую распаковали архив, запустите файл Setup.exe и  следуйте инструкциям программы-установщика.

3. Создать резервную копию информационной базы

Запустите 1С: Предприятие. В открывшемся окне Запуск 1С: Предприятия определите и откройте папку, в которой находится нужная информационная база. Затем скопируйте файл 1Cv8.1CD в безопасное место.

4. Начало работы с новой версией платформы

По кнопке 1С:Предприятие можно начать работы с выбранной информационной базой на новой версии платформы.
