Предназначена для настройки экспорта замеров производительности вручную для анализа в других программах.

В программе производится оценка производительности. Замеры производительности хранятся в программе и могут быть экспортированы с целью анализа в других программах.

С помощью кнопок  заполните поля Дата начала и Дата окончания экспорта замеров производительности.

Выберите профиль ключевых операций в поле Профиль.

Выберите путь для сохранения данных в поле Каталог экспорта.

Напишите Имя архива для сохранения замеров.
