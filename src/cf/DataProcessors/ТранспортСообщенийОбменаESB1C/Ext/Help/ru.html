Для настройки подключения через 1С:Шина необходимо указать следующие параметры:

  • Сервис интеграции, через который происходит подключение к 1С:Шине.
  • Канал получения для входящих сообщений обмена.
  • Канал отправки для исходящих сообщений обмена.
  • Код получателя в шине (код информационной базы, который указывается в шине).

Перед настройка синхронизации в 1С:Предприятии необходимо настроить процесс интеграции в 1С:Шина. Подробную информацию о создании процесса интеграции можно посмотреть по ссылке.

Добавьте новый процесс интеграции, со схемой как на картинке ниже. (имена блоков могут отличаться от имен в примере)

Добавьте инфосистемы. Для каждого приложения (базы 1с) необходимо добавить отдельную инфосистему.

Для каждой инфосистемы получите ключ API.

В конфигураторе базы 1с необходимо добавить новое расширение, а в нем новый сервис интеграции (это можно сделать и в основной конфигурации, если она снята с поддержки).

В сервис интеграции загрузить каналы, при помощи полученного ранее ключа API.

В модуле сервиса интеграции, в каждый обработчик необходимо добавить вызов процедуры Обработки.ТранспортСообщенийОбменаESB1C.ПоместитьСообщениеВХранилище(Сообщение, Отказ)

После сохранения расширения (конфигурации), необходимо перейти в режим предприятия и начать настройку синхронизации.

Через форму настройки подключения к шине включите регламентное задание и подключите сервис интеграции, используя ключ API.

После ввода настроек подключения к шине, продолжите настройку синхронизации следуя по этапам помощника создания синхронизации.
