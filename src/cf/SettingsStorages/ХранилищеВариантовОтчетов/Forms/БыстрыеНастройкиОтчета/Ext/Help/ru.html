Размещается в шапке отчетов.
  • В панель быстрых настроек выводятся, как правило, наиболее важные настройки: фильтры, оформление, группировки и т. д., некоторые из них могут быть обязательными для заполнения.
  • С помощью кнопки  можно открыть быстрые настройки и изменить их состав с помощью флажков.
  • С помощью кнопок настройки в панели можно менять местами.
  • С помощью кнопки Больше возможностей можно открыть Настройки отчета.
