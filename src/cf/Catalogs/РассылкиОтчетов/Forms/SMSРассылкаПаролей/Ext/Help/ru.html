Необходима для того, чтобы передать получателям их пароли к рассылке отчетов. Для связи с получателем можно выбрать любой доступный вид контактной информации (например, телефон, предназначенный для SMS-рассылки).

Пример SMS: Ваш пароль 12345 для получения рассылки "Демо:Персонализированная по получателям(HTML, NOZIP)".


После отправки SMS с паролями, отображается результат с комментариями. В случае возникновения ошибок при отправке, рядом с Получателем появляется символ предупреждения   например, предупреждает, что не указан номер телефона или не доступен.
