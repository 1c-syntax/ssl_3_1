Предназначен для хранения настроек подключения к сервисам облачной подписи (КриптоПро DSS)

 Сервис облачной подписи – это сервис на базе КриптоПро DSS предоставляемый провайдером. Данный сервис позволяет хранить сертификаты и закрытые ключи к ним в «Облаке». С помощью «облачных» сертификатов пользователь может задействовать функции подписания, шифрования, расшифровки данных практически с любого компьютера. Для использования этого функционала не требуется установки криптопровайдеров и сертификатов на локальный компьютер, достаточно предоставить данные для авторизации на настроенном сервисе.

 К каждому серверу облачной подписи привязываются реестр учетных записей этого сервиса, указываемые в справочнике «Учетные записи облачной подписи»
