Предназначен для хранения настроек подключения к серверам DSS (КриптоПро DSS)

В справочнике отражается информация о серверах DSS на базе КриптоПро DSS, предоставляемый провайдером. Данный сервис позволяет хранить сертификаты и закрытые ключи к ним. С помощью сертификатов пользователь может задействовать функции подписания, шифрования, расшифровки данных практически с любого компьютера. Для использования этого функционала не требуется установки криптопровайдеров и сертификатов на локальный компьютер, достаточно предоставить данные для авторизации на настроенном сервисе.

К каждому серверу DSS привязываются реестр учетных записей этого сервиса, указываемые в справочнике «Учетные записи DSS»
