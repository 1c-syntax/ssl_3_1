С помощью кнопки Видимость в подменю "Печать" можно ограничить доступность соответствующей команды печати в зависимости от значений реквизитов элементов. 
Видимость команд печати нужна для того, чтобы печатная форма могла быть сформирована только для подходящего документа.
Для ввода условия нажмите Добавить.
Выберите Поле документа, Вид сравнения, напишите или выберите Значение, с которым будет сравниваться поле.
