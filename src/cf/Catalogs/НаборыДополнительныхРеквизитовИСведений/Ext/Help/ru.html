Предназначен для настройки наборов дополнительных свойств.

Можно настроить наборы дополнительных свойств для определенных объектов программы. 

Способ хранения свойств в программе и место размещения в окне объекта программы зависит от типа свойства:

  • Дополнительные реквизиты - хранятся в самом элементе справочника или документе (могут размещаться в специальном разделе или на специальной вкладке). Создаются для реализации таких свойств, которые являются неотъемлемой частью объекта, вводятся при его редактировании и доступны для редактирования тем же пользователям, которым доступен и сам объект со свойствами.
  • Дополнительные сведения - предназначены для хранения сведений об элементе справочника или документе, которые не являются его неотъемлемой частью. Доступны, как правило, для просмотра и/или редактирования пользователям, которые не имеют доступа к самому объекту. Редактируются в отдельном окне, которое можно открыть из окна объекта.
  • Метки - предназначены для визуальной идентификации и классификации справочников или документов, помечая их разноцветными значками-метками. Редактирование и просмотр меток определены правами на объект-владелец.

Для разных объектов программы может быть предусмотрено использование свойств в разных сочетаниях: или дополнительных реквизитов, сведений и меток, или дополнительных реквизитов, или дополнительных сведений, или меток.

Для того чтобы использовать свойства, необходимо включить флажок Свойства в разделе Администрирование - Общие настройки - Свойства. После этого становятся доступными соответствующие команды.

Работа со свойствами осуществляется в списках, которые открываются по команде Дополнительные реквизиты, или Дополнительные сведения, или Метки в разделе Администрирование - Общие настройки - Свойства. 

Список наборов дополнительных реквизитов можно также открыть из объектов программы, для которых они настроены, по команде Еще - Изменить состав дополнительных реквизитов.

Список наборов дополнительных сведений можно также открыть по команде Еще - Изменить состав дополнительных сведений из окна Дополнительные сведения объекта программы.

Список Свойств представляет собой таблицу из двух частей:

  • В левой части располагаются наборы свойств, назначенных справочникам или документам, которые показаны в виде групп. Группы в наборах свойств соответствуют некоему состоянию объекта (его виду), например, для номенклатуры наборы по имени соответствуют видам номенклатуры.
  • В правой части в виде таблицы располагается список свойств, входящих в выделенный набор. Жирным шрифтом в списке дополнительных реквизитов отражаются реквизиты, обязательные для заполнения. Для дополнительных сведений и меток такая возможность не предусмотрена.
  • В таблице выводится:
    • Дополнительный реквизит (сведение) - включает перечень наименований свойств;
    • Тип значения - указывается тип свойств и количество значений.

Предопределенные наборы свойств соответствуют объектам программы (спискам или документам), в которых можно использовать свойства. Добавление новых наборов в этом списке невозможно. Наборы автоматически создаются при создании групп в списках.

Программа позволяет создать набор свойств, как для всех элементов справочника, так и для части. В этом случае справочник является группой, а элементами группы являются реквизиты справочника или значения реквизитов справочника (в этом случае количество элементов группы будет равно количеству значений реквизита).

Назначение нового свойства справочнику или документу

  • В зависимости от настройки может быть доступна одна из команд ввода новых свойств:
    • Выберите соответствующий этому справочнику или документу набор, нажмите Создать. По умолчанию свойство может быть добавлено только в один набор. Заполните необходимые поля.
    • Выберите соответствующий этому справочнику или документу набор, нажмите Добавить - Новый (Новое) - аналогична команде Создать. С помощью этой команды свойство может быть добавлено только в один набор. Заполните необходимые поля.

Общие наборы свойств

  • Для некоторых справочников предусмотрены предопределенные наборы Общие. Введенные в такой набор свойства по умолчанию становятся общими для всех элементов этих справочников.
  • Выберите набор Общий.
  • Нажмите Создать или Добавить - Новый (Новое). Заполните необходимые поля.

Добавление из другого набора

  • В программе предусмотрена возможность использовать ранее введенные свойства для других наборов.
  • Наиболее актуален этот способ для свойств типа Дополнительное значение и Дополнительное значение (иерархия).
  • Выберите набор свойств, нажмите кнопку Добавить, в выпадающем меню выполните команду Из другого набора.
  • Открывается помощник по добавлению свойств.
    • В левой части первой страницы помощника отображается список ранее введенных свойств. В правой части отражаются значения и другие параметры выбранного реквизита.
  • На первой странице помощника выберите свойство. Обратите внимание, что создать свойство в данном списке невозможно.
    • Для выбора варианта добавления свойства нажмите кнопку Далее.
    • Также можно добавить реквизит из одного набора в другой с помощью перетаскивания мышью. Если были произведены соответствующие настройки, открывается вторая страница помощника Добавление свойства.
  • На второй странице помощника с помощью переключателя выберите вариант добавления свойства:
    • Добавить реквизит как есть (рекомендуется) - сделать свойство общим. В этом случае общим становится не список значений, а сам реквизит.
    • Сделать копию реквизита по образцу - создается свойство с общим списком значений. 
    • Сделать копию реквизита - в новом наборе будет создана копия свойства и всех его значений.

Добавление свойств как есть

  • Общий реквизит входит в несколько наборов. Этот вариант подходит для тех случаев, когда общим должен быть не только список значений, но и сам реквизит. В списках и отчетах по элементам различных справочников и документов общие реквизиты группируются и выводятся как один реквизит в одной колонке или поле. Например, с помощью общего реквизита Цвет можно отобрать всю желтую одежду и обувь. 
  • Для того чтобы добавить в набор общее свойство, выделите нужный набор, нажмите Добавить - Из другого набора.
  • Выберите из предлагаемого списка уже существующий общее свойство. Нажмите кнопку Далее. 
  • Открывается вторая страница помощника. Выберите с помощью переключателя вариант добавления Сделать реквизит общим и добавить в набор, нажмите Далее. Открывается общий дополнительный реквизит. 
  • Также можно добавить общее свойство с помощью перетаскивания мышью или кнопок Копировать и Вставить.
  • Для того чтобы добавить общий реквизит в несколько наборов, повторите эти действия для каждого набора.

Добавление по образцу

  • Можно использовать ранее введенные свойства из других наборов, добавлять реквизиты с общим списком значений сразу для нескольких однотипных реквизитов (например, Цвет двери по образцу реквизита Цвет). Список значений в этом случае используется из реквизита-образца. Удобно выполнять централизованную настройку списка значений сразу для нескольких однотипных реквизитов. 
  • Выберите набор из списка.
  • Нажмите Добавить - Из другого набора.
  • Будет открыт список всех свойств, уже существующих в программе. Выберите свойство из списка уже существующих в программе, нажмите Далее.
  • С помощью переключателя выберите способ создания элемента Сделать копию реквизита по образцу, нажмите Далее.
  • Тип и все значения выбранного в качестве образца свойства будут скопированы в новый элемент. 
  • Заполните все необходимые поля. По умолчанию реквизит становится общим с набором, из которого был использован образец. Ввод новых значений в этом случае невозможен. Программа выводит напоминание, что список значений является общим с другим реквизитом.
  • После сохранения свойства с помощью ссылки можно просмотреть реквизит-образец и добавить значения, если необходимо.

Копирование свойств

  • В программе предусмотрена возможность копирования свойств.
  • В этом случае для набора свойств будет создана копия - самостоятельное свойство со своим списком значений, никак не связанный с оригиналом, по которому был создан.
  • В зависимости от настроек производится несколькими способами:
    • В помощнике добавления свойств;
      • Выделите набор дополнительных реквизитов в списке, нажмите Добавить - Из другого набора.
      • На первой странице помощника в списке выбора реквизитов для копирования выберите нужный реквизит, нажмите Далее.
      • На второй странице помощника с помощью переключателя выберите Сделать копию реквизита, нажмите Далее.
      • Заполните недостающие поля. После копирования реквизита список значений доступен для редактирования.
    • перетаскиванием мышью;
      • Можно перетащить нужное свойство из одного набора в другой с помощью мыши.
      • Если в списке имеется кнопка Добавить, то открывается вторая страница помощника добавления свойства.
      • Если в списке только кнопка Создать, то открывается карточка свойства, которая будет скопирована. Заполните недостающие поля.
    • с помощью кнопок Копировать и Вставить.
      • Выделите свойство, нажмите кнопку Копировать. Выделите набор, в который нужно добавить свойство. Нажмите кнопку Вставить. Если были произведены соответствующие настройки, то открывается вторая страница помощника добавления свойства. Если выбранное свойство уже входит в какой-либо набор, то предлагается сделать его общим.
      • Команды Копировать и Вставить также доступны в контекстном меню по правой кнопке мыши или в меню Еще.

Особенности удаления свойств

  • Для того чтобы удалить общее свойство, необходимо удалить его из всех наборов, в который входил данный реквизит. Возможно, для окончательного удаления придется очистить дополнительные реквизиты от значений.
  • При пометке на удаление свойства в наборе значения свойства сохраняется в объектах до тех пор, пока не будут очищены. Свойство, помеченное на удаление в наборе, будет отображаться в окне объекта или окне дополнительных сведений особым образом.
  • При необходимости можно снять пометку удаления.
