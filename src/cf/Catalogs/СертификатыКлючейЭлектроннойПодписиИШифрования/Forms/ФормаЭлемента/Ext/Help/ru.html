Сведения о сертификате электронной подписи.

В верхней части приложение автоматически выводит данные о сертификате:

  • Кому выдан - личные данные владельца сертификата;
  • Кем выдан - данные об УЦ, который выдал сертификат;
  • Действителен до - минимальная дата из срока действия сертификата и срока действия закрытого ключа. Создать подпись можно до этой даты. Используется при проверке подписи, если подпись не содержит метки времени;
  • Назначение - виды и коды назначений сертификата (подписание, шифрование).

Сведения о владельце сертификата:

  • Данные владельца подписи (заполняются автоматически) в поле Кому выдан:
    • Фамилия, Имя, Отчество;
    • Фирма;
    • Должность;
  • Представление - полное наименование сертификата (заполняется автоматически);
  • Заполните поля:
    • Организация - выберите из списка организацию, которой выдан сертификат;
    • Физическое лицо - выберите из списка физическое лицо или нажмите Подобрать автоматически  (физическое лицо подберется по наименованию из поля сертификата Кому выдан);
    • Пользователи - заполняется автоматически при добавлении сертификата. Редактировать можно в карточке сертификата. Выберите из списка пользователей. В карточке сертификата поле предназначено для указания списка пользователей, у которых данный сертификат будет отображаться в окнах подписания и шифрования данных;
    • Добавил - выберите пользователя, который добавил сертификат в список.

Настройки закрытого ключа

При необходимости можно хранить пароль не в приложении 1С:Предприятие, а в приложении электронной подписи.
    • Приложение - выберите приложение для электронной подписи и шифрования, в которой будет храниться пароль;
    • Ввод и сохранение пароля в приложении - включите флажок, если необходимо хранить пароль в приложении электронной подписи.

Проверка сертификата

  • Нажмите Проверить, для того чтобы произвести проверку сертификата. Проверка занимает некоторое время, проводится по нескольким параметрам. Если сертификат недействителен, то приложение выводит подробное сообщение о возможных причинах его недействительности. 

Просмотр данных сертификата

  • Нажмите , для того чтобы просмотреть данные сертификата.

Просмотр заявления

  • Нажмите , для того чтобы просмотреть заявление, по которому был получен сертификат.

Выгрузка в файл

  • Нажмите Сохранить в файл, укажите путь к папке для сохранения в файл.

Автозаполняемые реквизиты

  • Установите соответствующий флажок меню Еще, для того чтобы Показать автозаполняемые реквизиты. В этом случае показываются данные владельца подписи, которые заполняются автоматически.

Отозванные сертификаты

  • Включите флажок Еще - Сертификат отозван. Действие необратимо.
