Предназначена для продления сроков действия (усовершенствования) электронных подписей вручную.
В программе предусмотрено использование электронных подписей для длительного хранения документов. Выбрать тип подписи можно в разделе Администрирование - Общие настройки - Электронная подпись.
Продление сроков действия (усовершенствование) выполняется для выбранного файла или в списке - указывается подписанный объект (количество подписанных объектов).
Обычная электронная подпись (CAdES-BES) имеет ограниченный срок действия, он зависит от срока действия сертификата электронной подписи, который длится около года.
Для долговременного хранения в электронную подпись вносится метка доверенного времени (подпись CAdES-T) или архивная метка времени (подпись CAdES-A). 
Если в программе применяются подписи с доверенной меткой времени, то можно продлить (усовершенствовать) выбранную подпись - добавить доверенную метку времени. 
Программа предлагает выбрать способы усовершенствования подписи:
  • Усовершенствовать до - Выберите способ усовершенствования (тип подписи) из списка с помощью переключателя.
  • Не усовершенствовать.
  • Добавить архивные метки времени к архивным подписям - включите флажок для продления архивных подписей.
