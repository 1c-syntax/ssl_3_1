Склонения представления выбранного элемента списка.

Открывается по ссылке Склонения из карточки элемента списка.

По умолчанию склонение проводится программой собственными средствами. 

При необходимости можно использовать альтернативный сервис склонения "Морфер". Включить использование сервиса можно в настройках программы.

Возможны неточности автоматического склонения.

Поэтому предусмотрена возможность исправить неточные склонения и сохранить правильный вариант в карточке элемента списка, для того чтобы в печатные формы выводились правильные варианты склонения.

Исправьте склонения вручную, нажмите кнопку ОК. Для сохранения склонений в карточке элемента списка нажмите кнопку Записать и закрыть.

Если исправленный вариант склонения не устраивает, нажмите кнопку Просклонять повторно.
