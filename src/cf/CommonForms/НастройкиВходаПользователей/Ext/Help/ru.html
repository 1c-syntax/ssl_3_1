Предназначена для настройки параметров входа в приложение для внутренних и внешних пользователей.

Содержит четыре вкладки:

  • Для пользователей;
  • Для внешних пользователей (если разрешен доступ внешним пользователям);
  • Дополнительно;
  • Запрещенные пароли (доступно в платформе 1С:Предприятие 8.3.26).

Требования к паролю


Для защиты от несанкционированного доступа к приложению укажите с помощью флажков:

  • Пароль должен отвечать требованиям сложности - настройка и контроль сложности пароля. Включите флажок, для того чтобы проверять, чтобы новый пароль имел не менее 7 символов, содержал любые 3 из 4-х типов символов: заглавные буквы, строчные буквы, цифры, специальные символы, не совпадал с именем (для входа);
  • Минимальная длина пароля - по умолчанию 8 знаков;
  • Пароль должен отсутствовать в списке запрещенных - подробнее см. запрещенные пароли (доступно в платформе 1С:Предприятие 8.3.26);
  • Если одно из требований выше не выполнено при входе - то можно предложить или потребовать смену пароля (доступно в платформе 1С:Предприятие 8.3.26);
  • Максимальный срок действия пароля - срок после первого входа с новым паролем, после которого пользователю потребуется сменить пароль, по умолчанию 30 дней.
  • Минимальный срок действия пароля - срок после первого входа с новым паролем, в течение которого пользователь не может сменить пароль, по умолчанию 1 день.
  • Запретить повторение пароля среди последних - по умолчанию 10 значений. Таким образом, обеспечивается контроль повторяемости паролей;
  • Предупреждать об окончании срока действия пароля за - по умолчанию 5 дней до истечения срока действия пароля.  При входе в приложение появляется предупреждающее окно для смены пароля, где сразу можно сменить пароль;
  • Запрещать вход в приложение пользователям, которые не работали в приложении более - срок относительно последней активности пользователя, после которого вход в приложение будет запрещен, по умолчанию 45 дней.

Можно установить Отдельные настройки для внешних пользователей. 

На вкладке Дополнительно с помощью флажка Если пользователю ограничен срок работы в приложении, предупреждать за (по умолчанию 7 дней) - выводится предупреждение, если включено ограничение на вход в карточке пользователя или включена общая настройка входа.

Также отображается настройка Показывать в списке выбора в карточке пользователя: 

  • Включено для новых пользователей - в карточке нового пользователя флажок Показывать в списке выбора будет включен;
  • Выключено для новых пользователей - в карточке нового пользователя флажок Показывать в списке выбора будет выключен; 
  • Скрыто и включено для всех пользователей - при входе в приложение список выбора пользователей будет полным, а флажок Показывать в списке выбора в карточках всех пользователей будет включен и скрыт;
  • Скрыто и выключено для всех пользователей  - при входе в приложение список пользователей станет пустым, а флажок Показывать в списке выбора в карточках всех пользователей будет очищен и скрыт. 

При использовании платформы 1С:Предприятие версии 8.3.26 на вкладке Дополнительно также можно задать следующие параметры: 

  • Завершать сеанс при бездействии пользователя более  - если пользователь не выполняет никаких действий в приложении в течение определенного периода времени, то можно завершить его сеанс, по умолчанию более 960 мин. Также есть возможность Предупредить о завершении сеанса при бездействии, по умолчанию за 5 мин.; 
  • Разрешить сохранять пароль при входе в приложение - сохранение пароля при входе в приложение упрощает авторизацию и избавляет от необходимости вводить пароль каждый раз. При этом можно указать Время сохранения пароля, по умолчанию 7200 мин. 

В случае клиент-серверного варианта работы на вкладке Дополнительно также можно указать Количество попыток ввода пароля до блокировки (по умолчанию 3 попытки) и Длительность блокировки ввода пароля (по умолчанию 5 мин.).

На вкладке Запрещенные пароли отображается настройка списка запрещенных паролей, которая помогает повысить безопасность учетных записей пользователя (доступно в платформе 1С:Предприятие 8.3.26): 

  • Стандартный список - база паролей, которая встроена в платформу 1С:Предприятие;
  • Дополнительный список - позволяет установить список паролей из текстового файла; 
  • Сервис проверки - позволяет проверить вводимые пароли на соответствие списку запрещенных паролей. Если вводимый пароль совпадает с одним из запрещенных, система отказывает в регистрации такого пароля. Максимальное время ожидания сервиса, по умолчанию 1 сек. Также можно Разрешить вход, если сервис не сможет выполнить проверку.
