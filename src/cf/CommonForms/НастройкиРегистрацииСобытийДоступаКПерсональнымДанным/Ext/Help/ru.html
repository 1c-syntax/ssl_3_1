Предназначена для настройки регистрации событий доступа к персональным данным (в соответствии с требованиями Федерального закона от 27.07.2006 N152-ФЗ "О персональных данных").

Открывается по команде Регистрация событий доступа к персональным данным из раздела Настройки пользователей и прав - Администрирование.

Для регистрации событий доступа к личным данным физических лиц администратору приложения необходимо установить флажок Личные данные.

Для регистрации доступа к паспортным данным установите флажок Паспортные данные. После этого приложение автоматически регистрирует события доступа к этим данным в списке событий.
