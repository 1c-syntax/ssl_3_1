Основные печатные формы являются подтверждающими, первичными документами для хозяйственной операции. Они определяют внешнее представление документов.

Основная печатная форма по документу может запоминаться только одна. Поэтому при формировании комплекта из нескольких основных, необходимо выбрать какую запоминать.

Сохраненное наименование может быть использовано при формировании исходящих документов.
