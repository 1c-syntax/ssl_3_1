Предназначена для просмотра основных свойств сертификата ключа электронной подписи и шифрования.

Основные сведения о сертификате

  • Кому выдан - личные данные владельца сертификата;
  • Кем выдан - данные об Удостоверяющем центре, который выдал сертификат;
  • Действителен до - дата, до которой действует сертификат;
  • Назначение - виды и коды назначений сертификата:
    • Подписание - флажок включен, если сертификат предназначен для подписания;
    • Шифрование - флажок влючен, если сертификат предназначен для шифрования.
  • Субъект - список свойств субъекта сертификата в привычных названиях, которые удалось распознать;
  • Издатель - список свойств издателя сертификата в привычных названиях, которые удалось распознать;
  • Внутреннее содержание - состав и значения свойств, как они указаны в данных сертификата.

Выгрузка сертификата

  • Нажмите Сохранить в файл и укажите папку, для того чтобы сохранить данные сертификата на компьютер.

Проверка сертификата

  • Нажмите Проверить, для того чтобы проверить действительность сертификата.
