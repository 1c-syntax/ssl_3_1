Предназначен для вывода родительских и подчиненных объектов приложения.

Формируется командой Связанные документы на панели навигации объектов приложения.

Отчет предоставляет оперативный и наглядный доступ ко всей цепочке взаимосвязанных с точки зрения бизнес-логики документов и списков. Объекты приложения считаются взаимосвязанными, если один объект ссылается на другой. В приложении взаимосвязь объектов выглядит так: в одном объекте приложения (назовем его дочерним) в качестве какого-либо реквизита указан другой объект (назовем его родительским). Отчет позволяет из какого-либо одного объекта приложения увидеть всю цепочку взаимосвязанных с ним объектов.

В отчет попадают все дочерние и все родительские объекты текущего объекта приложения. В наглядной форме представлена взаимосвязь этих объектов.

Объект приложения, для которого сформирован отчет, отображается жирным шрифтом. Дерево родительских объектов строится вверх-вправо от объекта, для которого формируется отчет. Дерево дочерних объектов строится вниз-вправо от объекта, для которого формируется отчет.

Доступные действия

  • С помощью двойного щелчка мышью на любом выведенном в отчет объекте можно открыть данный объект для просмотра.
  • С помощью кнопки Обновить можно переформировать отчет.
  • С помощью кнопки Вывести для текущего можно сформировать отчет Связанные документы для выделенного объекта.
