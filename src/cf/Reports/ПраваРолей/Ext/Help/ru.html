Предназначен для технического специалиста, показывает права ролей на объекты метаданных (шахматка объект метаданных/роль: право), а также расшифровки этих прав. Не для базовых версий.
Отчет можно открыть:
  • в панели Отчеты администратора, ее можно найти в разделе Администрирование - Обслуживание;
  • из списка Профили групп доступа;
  • в Профиле групп доступа;
  • в списках по команде Еще - Отчеты (рядом с командой Права пользователей).
В настройках отчета можно добавить или убрать различные разделы или объекты программы, для того чтобы просмотреть права ролей и профилей на эти объекты.
С помощью кнопки Подбор открывается окно с перечнем объектов программы. Они обозначены в виде картинок или значков. При расследовании недостатка прав необходимо учитывать, что в сообщениях об ошибке программа выводится полное имя таблицы (например, Справочник.УдалитьНалоговыеОрганы), а в отчете в списке выбора выводится представление (например, (не используется)Налоговые органы).
  • Дважды кликнув по объекту, можно просмотреть расшифровку объекта метаданных - права ролей.
  • Дважды кликнув по роли, можно просмотреть расшифровку прав роли на нужный объект.
Из списка профилей в отчете можно просмотреть:
  • Права профилей (отбор по выделенным профилям);
  • Права ролей профилей (отбор по ролям выделенных профилей).
Из карточки профиля в отчете можно просмотреть:
  • Права профиля (отбор по профилю);
  • Права ролей профиля (отбор по ролям профиля).
Из списков в отчете выводится:
  • права ролей и профилей (с отбором по таблице).
