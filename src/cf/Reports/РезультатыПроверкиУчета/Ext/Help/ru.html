Предназначен для просмотра и устранения выявленных проблем с документами, регистрами и любыми данными в программе, для просмотра результатов проверок ведения учета.

В отчете выводится:

  • Вид проверки - сведения о проверке и перечень проблемных объектов;
  • итоговая колонка.

Сведения о проверке выделены цветом. В этой части выводятся:

  • Наименование проблемы и ее важность;
  • Возможные причины;
  • Рекомендации по исправлению;
  • Решение - в некоторых проверках может быть предусмотрена ссылка, по которой можно перейти к автоматическому исправлению проблемы.

Затем следует перечень обнаруженных в ходе этой проверки проблемных объектов программы, который состоит из колонок:

  • Данные о проблемном объекте;
  • Обнаруженная проблема;
  • Дата проведения проверки.

Отборы и настройки

  • В шапке отчета предусмотрены быстрые отборы по периоду, правилу проверки, ответственному, важности проблемы. 
  • Управлять отборами (добавить, включить, выключить) можно в шапке отчета и с помощью кнопки . 
  • С помощью флажка Выводить ответственного:
    • в отчете выводится колонка Ответственный; 
    • в специальных колонках по каждому ответственному выводится, сколько проблемных объектов обнаружено.
  • С помощью кнопки Настройки можно перейти к более сложным отборам (предусмотрен расширенный режим).

Просмотр объекта

  • Дважды щелкните по объекту для перехода к просмотру. Можно исправить обнаруженные в ходе проверки проблемы.
