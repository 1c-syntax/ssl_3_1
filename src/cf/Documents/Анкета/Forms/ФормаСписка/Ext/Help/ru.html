Предназначен для контроля хода проведения опроса.

Сотрудник, ответственный за проведение опроса, имеет возможность в реальном времени отслеживать и контролировать ход анкетирования. Заполненные респондентами анкеты попадают в список.

Список открывается по команде Анкеты раздела Анкетирование.

В списке выводится:

  • Дата - дата и время завершения редактирования анкеты респондентом.
  • Номер - номер анкеты, присваивается программой автоматически.
  • Опрос - информация о документе Назначение опросов.
  • Респондент - наименование Внешнего пользователя - респондента.

Просмотр анкеты

  • Можно открыть для просмотра анкету, например, с целью проверки полноты ее заполнения респондентом. Для этого воспользуйтесь двойным щелчком мыши.
