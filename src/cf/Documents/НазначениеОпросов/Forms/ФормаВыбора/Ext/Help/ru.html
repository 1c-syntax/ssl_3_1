Предназначен для выбора документа Назначение опросов.

Открывается по кнопке выбора из отчетов Анализ опроса и Аналитический отчет по анкетированию раздела Анкетирование.

В списке выводится:

  • Номер - автоматически присваивается программой;
  • Дата - дата и время создания документа Назначение опроса;
  • Наименование - полное наименование документа.

Выбор документа Назначение опроса

  • Выделите нужный документ, нажмите Выбрать.
  • Также можно воспользоваться двойным щелчком мыши.

Отбор по периоду

  • Выполните команду Еще - Установить интервал дат, выберите нужный период для просмотра документов.

Назначение нового опроса

  • Нажмите Создать, заполните необходимые поля.

Создание нового опроса

  • По команде Провести создается новый опрос с учетом установленных параметров. У респондентов появляется возможность заполнить анкету по данному опросу.

Отмена опроса

  • Выполните команду Еще - Отмена проведения, для того чтобы отменить опрос. После этого респонденты не увидят анкету по данному опросу.
